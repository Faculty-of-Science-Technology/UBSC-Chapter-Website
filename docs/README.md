## ğŸ¯ Project Overview

The UBSC Chapter Website is a comprehensive management platform for the University of Belize ACM Student Chapter. It provides:

- **User Management**: Registration, authentication, and role-based access control
- **Content Management**: Blog posts and event management
- **Group/Committee Management**: Organize members into groups
- **Invite System**: Controlled user registration through invite codes
- **Multi-language Support**: English and Spanish internationalization
- **Theme Management**: Customizable theme settings
- **Media Management**: Upload and manage media files
